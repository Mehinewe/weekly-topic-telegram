Generated badge avatars are written here automatically.

When the Monday poster runs, for each winner it builds
    <award_key>-<telegram_user_id>.png
(the winner's profile photo with the award emblem in the corner) and commits it
here, so GitHub Pages can serve it. The award's "Get Your Badge Avatar" button
links to docs/avatar.html?img=avatars/<that file>, where the winner downloads it.

These files are regenerated each week; old ones can be deleted safely.
