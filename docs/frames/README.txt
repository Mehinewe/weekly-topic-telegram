Optional real badge artwork goes here (one transparent PNG per badge type).

File names must match the badge_type in ../../awards.csv:

    shark.png
    microphone.png
    butterfly.png

Each should be a SQUARE PNG with a transparent centre (the ring/border/decoration
only) — it is drawn on top of the user's circular photo at full canvas size.

If a matching PNG is NOT here, the mini app falls back to a simple drawn coloured
ring + emoji medallion, so everything works without art. Add these later to make
the avatars look custom.
